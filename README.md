## myant (miya-ant) 跨平台组件库（微信小程序、支付宝小程序、H5）

### 安装

```
yarn add @myant/uni -S
或
npm install @myant/uni -S
```

### 开始

集成方式参考 uni-app 第三方 UI 库集成方法，eg:

```html
"easycom": { 
    "autoscan": true, 
    "custom": { 
        "^my-(.*)": "@myant/uni/lib/my-$1/index.vue" 
    } 
}
```

### 浏览器支持

现代浏览器以及 Android 4.0+, iOS 8.0+。
